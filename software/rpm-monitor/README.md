# RPM Monitor

Code for reading engine RPM by tapping the signal off the spark amplifier
(igniter), as a step toward a digital tach/speedo display.

## Status
🚧 In progress — circuit diagram and firmware to be added.

## Planned contents
- `src/` — microcontroller code for pulse counting / RPM calculation
- `hardware/` — circuit diagram and wiring notes for the spark-amp tap

## Notes

  comment block at the top describing the wiring (which pin taps which wire).
